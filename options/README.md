## Pipeline action option summaries

The `options` folder carries the `--help` output
of the different __TOOL_SUITE_NAME__ pipeline actions, including
complete documentation of all available options.

File names reflect the pipeline and action they
correspond to, e.g., `<pipeline>_<action>.yml`
is the output of `__TOOL_SUITE_SHORT_NAME__ <pipeline> <action> --help`.
